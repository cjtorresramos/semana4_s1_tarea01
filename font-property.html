Las propiedades de fuentes sirven para modificar distintos valores de la fuente.

Por ejemplo con la propiedad line-height se modifica el interlineado:

h2{ line-height: 1; }

Por ejemplo con la propiedad text-transform se puede transformar a minúsculas o mayúsculas, pero en este caso a minúsculas el texto:

h2{ text-transform: lowercase; }

Por ejemplo con la propiedad text-align en este caso se centra el texto:

h2{ text-align: center; }

Por ejemplo la propiedad font-weight se utiliza para colocar cuan negrita o ligera debe estar:

h2{ font-weight: 50; }

Por ejemplo la propiedad font-style se utiliza para especificar el estilo de fuente, en este caso cursiva:

h2{ font-style: italic; }

Por ejemplo la propiedad text-decoration se utiliza para los subrayados, en este caso para retirar el subrayado de un enlace:

a{ text-decoration: none; }
