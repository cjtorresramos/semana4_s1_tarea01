Tamaños de Fuentes

  • Familias Tipográficas
  • Tamaños de Fuentes
  • Propiedades de Fuentes

Los tamaños de fuente son las medidas de las letras.

Por ejemplo de Tamaño Absoluto:

h2{ font-size: 20px; }

Por ejemplo de Tamaño de Contexto:

h2{ font-size: 10em; }

Por ejemplo de Tamaño del HTML:

h2{ font-size: 10rem; }
